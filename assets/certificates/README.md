Place certificate JPG files in this folder using these names:

- microsoft-azure-ai-solutions.jpg
- gemini-certified-student.jpg
- commvault-pratidhi-gold-award.jpg

You can also add a JSON credential file and reference it from the certificate
card with `data-json="assets/certificates/file-name.json"`.

If you prefer online certificate links, edit the matching certificate card in `index.html`
and paste the URL into its `data-certificate-link` attribute.
